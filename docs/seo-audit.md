- Каноникалы: отсутствуют.

## 5) Индексация и robots
- Robots: файл и meta отсутствуют, значит индексация разрешена по умолчанию.
- `/thank-you` сейчас индексируем (нет `noindex`).
- Заголовки и контент есть в HTML, не скрыты от SSR.

## 6) SSR/CSR риски для индексации
- Большая часть секций — client components (`"use client"`), но основной текст всё равно отрисовывается сервером (есть SSR HTML).
- В `RootLayout` стоит `suppressHydrationWarning`, что может скрывать гидрационные рассогласования.
- Тяжёлые визуальные эффекты (motion/FX) не блокируют появление текста, но влияют на производительность.

## 7) Прочие наблюдения
- Динамические кейсы SSG через `generateStaticParams` (хорошо для индексации).
- Отсутствуют `robots.txt`, `sitemap.xml`, canonical, OG/Twitter, schema — требуется внедрение.
