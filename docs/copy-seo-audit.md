## 6. Слабые места контента

1. **Hero**: оффер понятен, bullets хорошие. Можно усилить «для кого» (малый бизнес, агентства, продукт).
2. **Услуги**: структура «что → кому → результат → сроки» соблюдена. FAQ по услугам — кратки, можно расширить под семантику.
3. **Кейсы**: «Было → Сделали → Стало» есть. Метрики — цифры есть ✓.
4. **FAQ (главная)**: 4 вопроса — мало для rich results. Добавить 4–6 вопросов из seo-keywords-ru.
5. **CTA ритм**: SectionCTA в Process, Integrations, WhyUs, CTA. Часть заменить на текстовые ссылки («Подробнее в кейсах», «Смотреть услуги»).

## 7. Рекомендации

1. Добавить `/demo` в sitemap.
2. Расширить FAQ (главная) — 6–8 вопросов под коммерческие кластеры.
3. Уменьшить плотность CTA: в 1–2 секциях заменить SectionCTA на текстовые ссылки.
4. Проверить, что title ≤ 60 символов, description 140–170.
5. Добавить перелинковку: Home → услуги + 3 кейса + 2 статьи; услуга → кейсы + статьи; кейс → услуга + похожие.
