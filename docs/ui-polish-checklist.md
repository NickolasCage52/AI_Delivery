# UI Polish Checklist (Desktop + Mobile)

Цель: не “уменьшенная копия”, а продуманный UX на 390 / 768 / 1024 / 1440.

---

## Брейкпоинты

- **390 (mobile)**: читаемость, тап‑таргеты ≥ 44px, отсутствие наложений, формы, sticky не перекрывает контент
- **768 (tablet)**: сетка, интервалы, карточки, меню
- **1024 (desktop)**: hover‑состояния, плотность, z‑index, фон/blur
- **1440 (wide)**: max‑width, line‑length, balance, отсутствие “пустоты”

---

## Desktop

- **Сетка/ритм**: единые отступы секций, логика заголовок → подзаголовок → контент
- **Типографика**: H1/H2/H3 — `--font-display`, body — `--font-sans`, тех‑лейблы — `--font-mono` (дозировано)
- **Hover states**: понятные, без “перекрытий” текста; интерактивные карточки не должны ломать layout
- **Z‑index**: модалки/меню/липкие элементы поверх фона и FX, без “пробивания”

---

## Mobile

- **Tap targets**: кнопки/меню/крестики ≥ 44px
- **Sticky CTA**: не перекрывает форму и ссылки, появляется не сразу (после 1–2 экранов)
- **Модалки/меню**: scroll lock, фокус внутри, Escape закрывает
- **Формы**: минимум полей, удобный ввод, понятные placeholders
- **FX**: не мешают чтению (opacity/контраст), нет лагов при скролле
